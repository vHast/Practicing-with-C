### 02 - Escape Sequences

Commenting follows the next format: 

```c
#include <stdio.h>

int main() {
 
	// This is a comment

 	/*
	This is a multiline comment
 	*/

 	printf("I like pizza!\n");
  	return 0;
}
```

Escape sequences specify actions within a line of text

```c
\t stands for tab
\n stands for newline

printf("\"I like Pizza\" - Abraham Lincoln");
```
